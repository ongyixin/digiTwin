- Breaking changes (BREAKING CHANGE, feat!, refactor!, etc.)

{commits}

{
  "decisions": [
    {
      "title": "Short title for the decision (under 80 chars)",

- Only include commits that represent genuine architectural decisions (confidence > 0.5)
